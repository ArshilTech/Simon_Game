# 🧠 Simon Game

A fun and interactive memory-based game built using **HTML**, **CSS**, and **JavaScript (jQuery)**.  
Test your memory and reflexes by repeating the color sequence shown by the game. Each correct level increases the difficulty!

---

## 🔗 Check it out: https://arshiltech.github.io/Simon_Game

---

## 🎮 How to Play
1. Press any key to start the game.
2. Watch the sequence of flashing colors carefully.
3. Click the colored buttons in the **same order** as they appeared.
4. If you succeed, the sequence grows longer with each level.
5. If you make a mistake — the game ends, and you can restart by pressing any key again.

---

## 🧩 Features
- Smooth animations and sound effects for each color.
- Increasing difficulty with every level.
- Instant feedback on wrong moves.
- Retro-style visuals using the **Press Start 2P** font.

---

## 🖥️ Technologies Used
- **HTML5** – Structure of the game.
- **CSS3** – Styling and visual effects.
- **JavaScript (ES6)** – Game logic and interaction.
- **jQuery** – DOM manipulation and animations.
